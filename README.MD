# Реп для обучения Web-программирования на Python
## Основые комманды GIT

- git --version - версия GIT
- git init - создание файлов для репозитория
- git status - проверка статуса
- git add . - зафиксировать изменения всех файлов
- git commit -m "any text" - создать коммит
- git log - вывод последних действий
